
# Bot de Monitoramento de Bitcoin no Telegram

Este bot envia alertas sobre o preço e volume do Bitcoin, e também gera um gráfico simples.

## Como usar

1. Instale as dependências:
   pip install -r requirements.txt

2. Execute o bot:
   python bitcoin_bot.py

3. No Telegram, envie os comandos:
   /start  - Inicia o bot
   /alert  - Verifica preço, volume e envia gráfico

## Requisitos

- Python 3.8 ou superior
- Conta no Telegram com um bot criado no BotFather
- API da CoinMarketCap
